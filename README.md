# InAmigos Foundation Awareness Webpage

## Overview

This project is a responsive NGO awareness webpage developed as part of the InAmigos Foundation Web Development Internship.

The webpage aims to spread awareness about the foundation's mission, ongoing projects, social impact initiatives, and volunteer opportunities through a clean and user-friendly interface.

---

## Features

* Responsive and modern design
* NGO introduction section
* Ongoing projects showcase
* Social impact section
* Mission in Numbers section
* Image gallery
* Contact information with clickable links
* Call-to-action buttons
* Smooth navigation experience

---

## Technologies Used

* HTML5
* CSS3

---

## Projects Highlighted

* Project Seva
* Project Bachpanshala
* Project Jeev
* Project Udaan
* Project Prakriti
* Project Vikas

---

## Objective

The objective of this webpage is to create awareness about the initiatives undertaken by InAmigos Foundation and encourage community participation through volunteering, support, and social engagement.
